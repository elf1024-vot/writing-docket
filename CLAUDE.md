# writing-docket

This repo is a Claude Code skill package. Install with:
```
claude skill install https://github.com/elf1024-vot/writing-docket
```

After install, the following skills are available:
- `/system-build` — Build a complete writing system from a plan document
- `/system-uninstall` — Tear down a project
- `/system-extend` — Add a genre extension to an existing project
- `/system-status` — Check health of a running project
- `/system-upgrade` — Upgrade to the latest skill version
- `/system-restore` — Restore from a backup
- `/novel-plan` — Load scene planning context

See README.md for full usage.
