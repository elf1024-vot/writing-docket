{{PROJECT_NAME}} is a writing docket: a local PostgreSQL-backed workspace for tracking characters, locations, chapters, and notes, with a web UI and a Claude MCP relay over the same data.

Quick Start

Run it

  1. Start Docker Desktop, then bring up the stack: docker compose up -d from {{PROJECT_PATH}}.
  2. Open the editors at http://localhost:{{UI_PORT}}.
  3. Fill in order: Locations → Characters → Chapters.
Claude

MCP Registration

claude mcp add {{PROJECT_SLUG}} python "{{PROJECT_PATH}}\mcp-server\mcp_relay.py"

Word Count

Running word_count.py

Recompute word counts from your manuscript files into the database:

python scripts\word_count.py

It scans Chapters/, counts words, and updates the read-only word_count column for each chapter/part.
